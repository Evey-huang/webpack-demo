# web pack-demo

学习web pack时为了更深刻地理解做的练习。webpack学习网站可以戳[这里](https://doc.webpack-china.org/)

### Getting start

```shell
$ git clone https://github.com/Evey-huang/webpack-demo.git
$ cd webpack-demo
$ npm install
$ npm run start
```
